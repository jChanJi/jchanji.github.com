前言

本篇主要讲解<? extends T> 和 <? super T>的一些基础概念，参考了传送门文章。

基本术语

  1. PECS原则

    PECS（Producer Extends Consumer Super）原则,频繁往外读取内容的，适合用上界Extends。经常往里插入的，适合用下界Super。

  2. 基类

    包含所有实体共性的class类型，即父类

  3. 派生类

    利用继承机制，新的类可以从已有的类中派生，即子类

  4. 其他术语(参考Effective Java)
