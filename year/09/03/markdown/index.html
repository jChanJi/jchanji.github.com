2、常用快捷键

  1. Ctrl + Alt + O //在浏览器中打开
  2. Alt + M //生成html文件
  3. Ctrl+Alt+O: Preview Markup in Browser.
  4. Ctrl+Alt+X: Export Markup as HTML.
  5. Ctrl+Alt+C: Copy Markup as HTML.
